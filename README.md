# Amrutam Telemedicine Backend

## Requirements
- Node.js v20+
- PostgreSQL
- npm

## Setup
1. Clone repo
2. Install dependencies: `npm install`
3. Set environment variables in `.env`
4. Run DB migrations (Sequelize auto-sync)
5. Start server: `npm run dev`

## Endpoints
- /api/auth/register
- /api/auth/login
- /api/doctor/*
- /api/booking/*
- /api/consultation/*
